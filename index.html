Recent Projects

Project 1

Conway's Game of Life

React CSS HTML Javascript

A modified version of Conway's Game of Life which is a simulation of a system where individual cells “live” and “die” in a state of relative equilibrium. After an in-game moment passes, the state of the game changes based on the previous state of the world. Users will be given the opportunity to modify some of the aspects of the game before starting the simulation. I wrote this based on a 2D grid system, where every individual location on the grid is refered to as a cell. I used React and React Hooks to build this web application with emphasis on state management and proper react principles such as making sure react components are small, reusable, and have a single purpose. Logic for determining whether cells become alive or dead was based on a brute force algorithm that checks all eight directions of a current cell. I also improved user experience by giving the user the ability to select the size of the board, speed of each iteration, start and stop the simulation, and manually revive or kill a cell. Finally, I also created a heatmap option which indicates how recently a cell was alive.

Project 6

Othello Board Game

Python Processing

This is a Reversi board game designed and implemented with Python on Processing, a flexible software sketchbook. I designed and implented classes that launched the setup of the 2D board, determined whose turn it would be, and flipped the disk colors when a valid move was made. I also designed a simple AI opponent that executed a DFS algorithm to find all possible moves during the current turn. I improved user experience by creating a leaderboard, letting the user know which tiles were flipped, and announcing the winner after every game. This was my first project on Python.
